# CMS
🖥️ PHP Content Management System (CMS)
A lightweight and efficient Content Management System (CMS) built using PHP and MySQL, designed for easy content creation, management, and customization. This project provides an intuitive interface for users to manage web content without requiring technical expertise.

✨ Features
✔️ User Authentication – Secure login and role-based access control
✔️ Content Management – Create, edit, delete, and organize posts and pages
✔️ Media Upload – Support for images and file uploads
✔️ Customizable Themes – Modular and easy-to-edit frontend design
✔️ SEO Optimization – Friendly URLs and metadata support
✔️ Database Integration – Efficient MySQL database management
✔️ Admin Dashboard – Interactive and user-friendly control panel

🚀 Technologies Used
Backend: PHP, MySQL

Frontend: HTML, CSS, JavaScript

Frameworks/Libraries: Bootstrap (for styling)

Security: Prepared statements, password hashing
